Block oder Inline?

Die Eigenschaft display bestimmt die Art und Weise, wie ein Element gerendert wird.
Das Standardverhalten ist abhängig vom HTML-Tag.
Die bekanntesten Werte sind: none, block, inline, inline-block
Weiter gibt es noch Tabellenspezifische Werte wie table, table-column-group, ...
Für Listen (li) der Wert list-item
Und für die modernen CSS-Layouts flex und grid

Ein Block-Element nimmt die gesamte zur Verfügung stehende Breite ein.
Zu den Block-Elementen gehören u.a.: div h1 h2 h3 p ul ol

Im Gegensatz zu den Block-Elementen nehmen Inline-Elemente nur soviel Platz wie nötig ein. Sie werden also innerhalb einer Reihe (inline) gerendert.
Beispiele: a span button i b img input label select textarea
