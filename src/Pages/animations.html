Animationen und Übergänge

Es gibt verschiedene Möglichkeiten Webseiten mehr Dynamik zu verleihen. Sehr beliebt sind kleine Animationen. Während bei Spielen und Unterhaltung bunte und bewegende Objekte im Vordergrund stehen, werden Animationen im professionellen Bereich oft nur zur Unterstreichung der Benutzerführung und für weichere Übergänge verwendet.

Für komplexe interaktive Anwendungen benötigt man weiterhin Javascript-Kenntnisse. Hier bietet sich zum einen die Canvas-API an. Für rechenintensive Grafiken und 3D-Szenen kann WebGL verwendet werden. Seit 2017 wird an einem neuen Standard für hardwarenahe Grafikberechnungen gearbeitet: WebGPU soll zukünftig WebGL ersetzen.

Für einfachere Animationen kann auch CSS eingesetzt werden. Mit etwas Kreativität lassen sich damit auch 'ohne' Programmierkenntnisse beeindruckende und 'bewegende' Webseiten gestalten.

Es gibt in CSS 2 Arten von Animationen:

1. Transitions

wird lediglich beim Setzen eines CSS-Wertes der alte Wert nicht einfach überschrieben sondern über einen definierten Zeitraum der Startwert in den Zielwert transformiert.

Hover Me
.mytransition: {
  transition-delay: time (s, ms)
  transition-duration: time (s, ms)
  transition-property: css-property (eg. width, height, color)
  transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(n,n,n,n)
  transition: prop duration timing-function delay
}

2. Animations

Mit CSS Animations lassen sich komplexere Animationen definieren. Hier können über mehrere Wegpunkte (Keyframes) einem HTML-Tag mehrere CSS-Attribute zugewiesen werden. Weiter kann die Animation vorwärts, rückwärts, mehrmals oder in Dauerschleife abgespielt werden. 

.myanimation: {
  animation-name: animation_name
  animation-play-state: running | paused 
  animation-iteration-count: n | infinite
  transition-delay: time (s, ms)
  animation-direction: normal | reverse | alternate | alternate-reverse
  transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(n,n,n,n)
  animation-duration: time (s, ms)
  animation: name duration timing-function delay iteration-count direction
}

@keyframes animation_name {
  from {property: value;}
    to {property: value;}
}
@keyframes animation_name {
    0% {property: value;}
    x% {property: value;}
  100% {property: value;}
}

2D und 3D Transformationen

Mit der Eigenschaft transform kann man Elemente 2-oder 3-dimensional transformieren. In Verbindung mit transition lassen sich nette Effekte erzielen

rotate(40deg)
scale(1.2)
skew(20deg, 40deg)
.transition_and_transform: {
  transition: transform 2s;
  transform: translate() | scale() | skew() | rotate() | matrix
  transform: rotate3d() | translate3d() | scale3d() | matrix3d
  perspective: 
  backface-visibility: hidden | visible
}
