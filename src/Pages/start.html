Einführung

Zur Beschreibung von modernen Webseiten werden die 3 Grundsprachen des Webs - HTML, CSS und Javascript - verwendet.

  • HTML: Hypertext Markup Language beschreibt die Struktur eines Dokuments
  • CSS: Cascading Style Sheets beschreibt, wie das strukturierte Dokument dargestellt werden soll
  • Javascript: Mit Javascript lässt sich die Struktur und des Dokuments dynamisch verändern.

Bevor es los geht

  • Projekt auschecken:
    git clone https://github.com/robindanzinger/css_workshop.git
  • In das Verzeichnis wechseln und Projekt bauen
    npm install
  • Projekt starten
    npm run dev
  • Webseite aufrufen
    localhost:3000/page/start.html
  • Die CSS-Dateien für die Übungen liegen hier:
    /exercises/css/
