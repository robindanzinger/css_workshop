CSS Selektoren

Aufbau einer CSS Datei

css-selector {
  css-property: value;
  css-property: value;
  css-property: value;
  ...
}
css-selector {
  css-property: value;
  ...
}
...

Eine CSS-Datei enthält also eine Anzahl an Regeln. Über den CSS-Selektor werden die Elemente im DOM ausgewählt und dann die CSS-Eigenschaften zugewiesen.

Beispiel: linke Seite die Struktur des Dokuments, mitte die CSS-Regel, rechts das Ergebnis 

  ...
  body
    main
      h1 Eine Überschrift
        div text="ein blaues [span class="hintergrundblau"]Wort[/span] "
        div text="und noch mehr Text"
  ...
          
h1 {
  color: red;
}
.hintergrundblau {
  backgroundcolor: lightblue;
}

Beispiel CSS-Selektoren

  • {tagname}
  • .{classname}
  • #{id}
  • *
  • div p
  • div, p
  • div > p
