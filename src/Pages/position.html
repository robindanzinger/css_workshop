Positionierung

Über die Eigenschaft position kann über die Eigenschaften top, right, bottom und left defniert werden, wie das Element positioniert wird.
Im Standardwert static wird das Element nach dem normalen Dokumentenfluß positioniert. Die Werte top, right, bottom, left und z-index haben dann keine Wirkung!
